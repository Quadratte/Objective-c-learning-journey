# Objective-C Learning Journey

> Мой годовой марафон по изучению Objective-C через классические книги.
> Старт: март 2026
> Финиш: март 2027

## Цель
Освоить язык Objective-C, научиться проектировать архитектуру приложений и написать несколько полноценных приложений (с поддержкой Apple Watch).

## Устройства для тестирования
- iPhone 5 (iOS 10.3.4) — основной полигон
- iPad 2 (iOS 9.3.5) — тест универсальных приложений
- Watch Series 3 (watchOS 8.8.1) - дополнительная функциональность
